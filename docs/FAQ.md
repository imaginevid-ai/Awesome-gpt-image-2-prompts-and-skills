# FAQ

## What is this repository?

This is a source-backed public collection of GPT Image 2 prompts, prompt skills, visual examples, and attribution metadata.

It is an independent community repository, not official OpenAI documentation. Model capability statements in the README should be tied to official OpenAI docs.

## What official GPT Image 2 facts does the README rely on?

OpenAI documents GPT Image models, including `gpt-image-2`, for image generation and image edits. The Image API exposes generations and edits; the Responses API supports image generation in conversational and multi-step flows, including multi-turn editing. Output can be adjusted by size, quality, format, and compression where supported.

## Where does the data come from?

Maintainers can collect candidates through local tools, manual curation, or community GitHub Issues. Approved entries are normalized into the same prompt data shape and rendered into README files.

## Does this repo require an external service?

No. README generation uses local structured data in this repository.

## How do I submit a prompt?

Use the **Submit a Prompt** GitHub issue template and include the prompt text, generated images, author, source link, and language.

## Why are there many README files?

The generated README set gives contributors and readers localized entry points while keeping the same source data and attribution rules. We prioritize the shared intro strings and obvious stale copy over hand-polishing every locale in every update.

## How do removal requests work?

Open an issue with the affected source link and the requested action. Maintainers will remove or update the entry.
